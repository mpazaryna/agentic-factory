# Chiro MLX

MLX model training pipeline for the chiro project.

## Components

### Skills
- **quality-checker** — Model quality validation
- **phi-guardian** — Phi model safety and compliance guardian
- **deployment-helper** — Model deployment assistance

### Commands
- **deploy-guide** — Deployment workflow guide
- **quality-check** — Run quality checks
- **status** — Pipeline status
- **train** — Model training workflows
- **validate-phi** — Phi model validation
