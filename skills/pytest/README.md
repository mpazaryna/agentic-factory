# pytest

> Run Python test suites and get structured results.

## What This Skill Does

- Runs pytest with verbose output and summarizes results
- Reports failures with test names and descriptions
- Confirms report generation if the project uses a report plugin
- Supports selective test runs via `-k` filters, markers, and file paths

## Quick Start

```
/test
"Run the tests"
"Run tests for the content cleaner"
"Run tests -k test_r11"
```

## When to Use

- After implementing a feature or fix
- Before committing to verify test health
- When you want a quick summary of what's passing and failing
