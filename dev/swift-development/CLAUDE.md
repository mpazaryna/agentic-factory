# Swift Development

Swift and SwiftUI development patterns, conventions, and submission workflows.

## Components

### Skills
- **swift-lang** — Swift language mastery: macros, concurrency, protocol-oriented design, performance
- **swift-ui** — SwiftUI implementation patterns: views, state management, navigation, animations
- **swiftui-submission-prep** — App Store submission preparation: screenshots, metadata, compliance checks

## How They Work Together

`swift-lang` provides the language foundation, `swift-ui` builds on it with framework-specific patterns, and `swiftui-submission-prep` handles the final mile of getting an app into the store. Use swift-lang for any Swift project; add swift-ui for UI work; add submission-prep when approaching release.

## Conventions
- Skills reference Apple's latest APIs and conventions
- Platform patterns should work across iOS, macOS, visionOS where applicable
- Submission prep assumes Xcode project structure
