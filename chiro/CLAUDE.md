# Chiro

Domain tools for the chiro iOS/macOS chiropractic application project.

## Components

### Agents
- **clickup-refiner** — Refines ClickUp tickets for the PAB app
- **config-auditor** — Audits PlatformConfig patterns against ADR-000
- **spec-writer** — Writes technical specs from PRDs for the PAB codebase

### Commands
- **install** — Project setup and installation
- **refine-ticket** — Trigger ticket refinement workflow
- **tk-agent** — ClickUp ticket agent operations
- **tk-close** — Close ClickUp tickets
- **tk-investigate** — Investigate ticket context
- **tk-open** — Open/create ClickUp tickets
- **uat-audit-cmd** — Run UAT audit
- **uat-new** — Create new UAT entries
- **uat** — UAT management
- **write-spec** — Generate technical specs
- **xcode/** — Xcode build, test, and management commands

### Skills
- **mlx** — MLX model integration patterns for the chiro app
