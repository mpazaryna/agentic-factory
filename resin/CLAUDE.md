# Resin

Domain tools for the Resin data extraction platform.

## Components

### Agents
- **clickup-refiner-resin** — ClickUp ticket refinement for Resin
- **spec-writer-resin** — Technical spec writing for Resin architecture

### Commands
- **extract-agent** — Run extraction agents
- **extract-worker** — Run extraction workers
- **mill** — Mill pipeline operations
- **refine-ticket-resin** — Ticket refinement workflow
- **run-agents** — Execute agent pipelines
- **run-enrich** — Run enrichment pipelines
- **run-specs** — Execute spec workflows
- **run-sync** — Run sync operations
- **ticket** — Ticket management
- **write-spec-resin** — Generate specs
