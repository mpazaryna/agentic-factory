# Install Agentic Factory Command

> Install skills, agents, or commands from the agentic-factory repo into any project

## Quick Start

```
/install-agentic-factory skill cloudflare
/install-agentic-factory agent research-agent
/install-agentic-factory command gh-issue
```

## Overview

The install-agentic-factory command copies components (skills, agents, commands) from your local agentic-factory repository into the current project's `.claude/` directory, making them immediately available.

## What This Command Does

- Parses component type and name from arguments
- Locates the agentic-factory repo via `$AGENTIC_FACTORY_HOME` or default path
- Copies the full component directory to `.claude/<type>s/<name>/`
- Verifies the installation and reports results

## Supported Component Types

- **skill** - Multi-file skill packages with SKILL.md, references, etc.
- **agent** - Specialized agent definitions with markdown configs
- **command** - Slash command files with YAML frontmatter

## When to Use

Use this command when:
- Setting up a new project with factory components
- Sharing a skill/agent/command across projects
- Installing a pre-built component for immediate use

- Claude Code with slash command support
- Local clone of the agentic-factory repository
- `$AGENTIC_FACTORY_HOME` env var set, or repo at `~/workspace/agentic-factory`
