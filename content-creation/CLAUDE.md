# Content Creation

Create content: blog posts, guides, UI design, documentation structures, and multi-source synthesis.

## Components

### Skills
- **writing** — SEO blog posts with structured approval workflow
- **compose-guide** — Compose guides from UAT walkthrough files into cohesive handbooks
- **frontend-design** — Production-grade frontend interface design (UI/CSS)
- **project-moc-generator** — Map of Content documentation generation

### Agents
- **synthesis-executor** — Combines multiple analysis outputs into executive summaries

## How They Work Together

`writing` and `compose-guide` produce written content. `frontend-design` produces visual content. `project-moc-generator` creates documentation structure. `synthesis-executor` pulls from multiple sources to create summaries — it pairs well with any of the content skills when you need to combine outputs.

## Conventions
- Written content follows SEO best practices when applicable
- Frontend design prioritizes production-ready code over mockups
- MOC generation follows established documentation patterns
