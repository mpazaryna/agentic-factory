# Yoga

Multi-agent yoga class planning system for teachers.

## Agents

- **orchestrator** — Routes yoga queries to specialist agents and synthesizes responses
- **asana-strategist** — Sequence architect: pose progressions, class structure, balanced practices
- **anatomy-expert** — Biomechanics, contraindications, modifications, anatomical rationale
- **theme-developer** — Teaching narratives, verbal cues, thematic arcs connecting poses to intention
- **professor** — Teaching readiness evaluator: scenarios, knowledge probes, scoring
