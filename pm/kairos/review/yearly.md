# Yearly Review Template

Writes to `kairos/logs/yearly/YYYY.md`. Create folder if needed.

Schedule: end of December.

## Pre-Flight

1. **Quarterly reviews** -- read `kairos/logs/quarterly/YYYY/YYYY-QN.md` for all four quarters. Primary source.
2. **Life-area notes** -- read all files in `kairos/life-areas/`
3. **Roadmaps** -- read `kairos/roadmaps/`
4. **Projects** -- read `projects/*.md` for final status
5. **Devlog** -- read `kairos/devlog/` for trajectory context

## Notes

- Yearly trusts quarterly reviews. Don't re-derive.

- Update goals.md with final status annotations
- This is a reflection, not a report card
- Hold space for what surprised you -- the plan is never the whole story
