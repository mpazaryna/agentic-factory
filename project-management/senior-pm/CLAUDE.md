# Senior PM

Portfolio management, quantitative risk analysis, resource capacity planning, and executive reporting for enterprise projects. Includes Python analysis scripts, reference frameworks, and reporting templates.

## Skills

- **portfolio-health** — Multi-dimensional health scoring with RAG status, trend analysis, executive recommendations
- **risk-analysis** — EMV calculations, Monte Carlo inputs, risk matrices, mitigation strategies
- **resource-planning** — Utilization modeling, skill matching, bottleneck identification, what-if scenarios

## Scripts

- `portfolio-health/scripts/project_health_dashboard.py` — Health scoring with RAG status
- `risk-analysis/scripts/risk_matrix_analyzer.py` — Risk quantification and EMV analysis
- `resource-planning/scripts/resource_capacity_planner.py` — Capacity modeling and scenarios

## Shared Assets

- `shared/assets/sample_project_data.json` — Sample multi-project portfolio data
- `shared/assets/executive_report_template.md` — Board-level portfolio report template
- `shared/assets/project_charter_template.md` — Comprehensive project charter template
- `shared/assets/raci_matrix_template.md` — Enterprise RACI matrix template
- `shared/portfolio-prioritization-models.md` — WSJF, RICE, ICE, MoSCoW frameworks
